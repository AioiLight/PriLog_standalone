■はじめに■
・この度はPriLogをダウンロード頂きありがとうございます。

・このソフトウェアは動画から
タイムラインの自動解析ができます。

・PC内に保存されている動画を対象として解析が可能です。

・YouTubeの動画を対象とした解析も可能です。


--------------------------------------------------------
■できること■
・スタンドアロンでのタイムライン解析ができます。
そのため、
動画を撮影　→　タイムライン解析
というフローを速く行うことができます。

・解析結果はWeb版から独立しています。
解析情報を作者が使用者に無断で収集及び保存を致しません。


--------------------------------------------------------
■動作環境■
・OS Microsoft Windows 10 
・ディスプレイ 解像度1024×768以上

・解析動画 解像度のアスペクト比が16:9の動画 (推奨解像度1280x720以上)


--------------------------------------------------------
■起動方法■
・フォルダ内にある"PriLog.exe"を実行してください。


--------------------------------------------------------
■アンインストール方法■
・フォルダごと手動で削除してください。
